## 函数
+ 函数是一种特殊的映射（一对一，一对多）
+ 函数是一个子过程
+ 命题逻辑与函数

```Haskell
F : A -> B
f(x) = ...

g: Integer -> Integer
g(x) = 2 * x + 1
```

## 程序
+ 程序设计：使用计算机解决现实问题
+ 程序设计语言：一种完备的，方便交与计算机执行的语言
+ 算法：解决问题的方法的描述
+ 程序：用程序设计语言把算法写出来
> 程序也可以是一个函数，对于给定输入数据给出输入数据，写程序要明白想要的输入数据和输出数据。

## 数据
+ 数据：采取符号表示信息
+ 基本数据类型：布尔/数值/字符
```Haskell
23    :: Integer
12.4  :: Float
True  :: Bool
"HJT" :: String
```

## 例子
```Haskell
bmi : 体重 身高 -> 胖瘦程度
bmi : (RealNumber, RealNumber) -> RealNumber
bmi : (w, h) = w / h ^ 2
```
